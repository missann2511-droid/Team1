Models Comparison

Best Model

Best model: Random Forest — lowest RMSE ($4,514), lowest MAE ($2,492), highest R² (0.7996).

Why RMSE as the main metric: the target is a monetary value (selling price in $), so an error metric in the same units that penalises large mistakes more strongly is the most relevant for pricing. Random Forest gave the best RMSE/R² trade-off while remaining robust to the many One-Hot categorical features and outliers, which is why it is selected and saved as the production model (models/best/best_model.joblib).
